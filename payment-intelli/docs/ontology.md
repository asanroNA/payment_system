# Ontology notes (FIBO mapping for Phase 2)

Human source of truth for the FIBO (Financial Industry Business Ontology) mapping that the
`semantic` block on every event references (CLAUDE.md §10).

Each event carries `SemanticMeta` (`entity_type`, `ontology_terms`, `glossary_ref`); the
`glossary_ref` points back into this file. Entries are added alongside each new event.
